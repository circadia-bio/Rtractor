# Rtractor continuous fill scale

Interpolates across an Rtractor palette for continuous `fill` data.

## Usage

``` r
scale_fill_rtractor_c(palette = "seq_blue", reverse = FALSE, ...)
```

## Arguments

- palette:

  Palette name. Default `"seq_blue"`.

- reverse:

  Logical. Reverse the palette? Default `FALSE`.

- ...:

  Additional arguments passed to
  [`ggplot2::scale_colour_gradientn()`](https://ggplot2.tidyverse.org/reference/scale_gradient.html).
