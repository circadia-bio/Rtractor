# List all available Rtractor palettes

Prints the names and sizes of all palettes defined in the package.

## Usage

``` r
rtractor_palettes()
```

## Value

A named integer vector of palette lengths, invisibly.

## Examples

``` r
rtractor_palettes()
#> Available Rtractor palettes:
#>   Qualitative:
#>     main                  5 colours
#>     core                  4 colours
#>   Diverging:
#>     diverging             6 colours
#>   Sequential:
#>     seq_blue              5 colours
#>     seq_sage              5 colours
#>     seq_coral             5 colours
```
